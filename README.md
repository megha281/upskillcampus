# 🎯 Python Quiz Game (CLI)

This is a command-line based Quiz Game developed using Python as part of an internship project. The application allows users to test their knowledge through multiple-choice questions with different categories and difficulty levels.

---

## 🚀 Features

* Multiple-choice quiz system
* Category selection (Python / General Knowledge)
* Difficulty levels (Easy, Medium, Hard)
* Timer-based questions
* Score calculation with percentage
* High score storage using file handling
* Replay option
* Input validation for user responses

---

## 🛠 Technologies Used

* Python (Core concepts)
* File Handling
* Object-Oriented Programming (OOP)

---

## ▶️ How to Run

1. Download or clone the repository
2. Navigate to the project folder
3. Run the program using:

```
python quiz_game.py
```

---

## 📌 Project Structure

* `quiz_game.py` → Main Python file
* `highscore.txt` → Stores highest score (auto-created)

---

## ⚠️ Note

* The timer checks response after user input (not real-time interruption)
* Make sure Python 3 is installed before running
